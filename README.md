# UI Design - Flutter Lab

**Course Code:** CS605PC  
**Department:** Computer Science and Engineering  
**Institution:** ACE Engineering College, Hyderabad

## Overview

This repository contains practical implementations for the UI Design - Flutter Lab course. The lab focuses on building cross-platform mobile applications using Flutter framework and Dart programming language.

## Course Objectives

- Learn installation of Flutter SDK, Xcode, and Android Emulator
- Understand Stateless and Stateful Widgets and Widget Tree
- Master Dart programming basics
- Apply animations to Flutter applications

## Getting Started

### Prerequisites

- Flutter SDK (latest stable version)
- Dart SDK
- Android Studio / VS Code
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/23-507/uiFlutter.git
   cd uiFlutter
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the application**
   ```bash
   flutter run
   ```

##  Lab Experiments

### Experiment 1: Introduction to Flutter and Dart
- Install Flutter and Dart SDK
- Write simple Dart programs
- Understand Flutter development environment

### Experiment 2: Flutter Widgets and Layouts
- Explore Text, Image, Container widgets
- Implement Row, Column, and Stack layouts
- Build complex UI structures

### Experiment 3: Responsive UI Design
- Design responsive layouts for different screen sizes
- Implement MediaQuery and breakpoints
- Create adaptive mobile, tablet, and desktop views

### Experiment 4: Navigation Between Screens
- Setup navigation using Navigator
- Implement named routes
- Handle screen transitions

### Experiment 5: Stateful & Stateless Widgets
- Understand widget lifecycle
- Implement state management with setState
- Use Provider for global state management

### Experiment 6: Custom Widgets and Styling
- Create reusable custom widgets
- Apply themes using ThemeData
- Implement custom styles

### Experiment 7: Forms and Validation
- Design forms with various input fields
- Implement form validation
- Handle error messages

### Experiment 8: Animations in Flutter
- Add fade and slide animations
- Use AnimationController
- Implement implicit and explicit animations

### Experiment 9: API Integration
- Fetch data from REST APIs
- Parse JSON responses
- Display data in ListView and Card widgets

### Experiment 10: Testing and Debugging
- Write unit tests for components
- Use Flutter debugging tools
- Implement test cases

##  Technologies Used

- **Flutter** - UI toolkit for cross-platform development
- **Dart** - Programming language
- **Provider** - State management
- **HTTP** - API integration
